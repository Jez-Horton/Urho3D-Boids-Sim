A simple game created in the Urho3D Engine, an open source games engine available on GitHub.

This project was created in the desire to understand more about boid simulations and it was then turned into a multiplayer game.

How to play: 
WASD Keys to move
Spacebar to shoot.
Hit the boids to gain points!

Singleplayer: Click the singleplayer button at launch.

Multiplayer, launch two instances of the game and run the server on one. The default ip is localhost.
